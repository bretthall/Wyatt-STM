Detailed Description

Functions that cannot be called from within a transaction should take one of these as an argument, with a default constructed object as the default value.

When this class is constructed it checks to see if a transaction exists and if it finds a transaction it throws WInAtomicError. Generally this is used via the NO_ATOMIC macro.
